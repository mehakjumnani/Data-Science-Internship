# Task 4 - Sales Prediction using Python

## Project Overview
This project predicts product sales based on advertising expenditure across TV, Radio, and Newspaper platforms using Linear Regression.

## Dataset Features
- TV
- Radio
- Newspaper
- Sales

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow
- Data Loading
- Data Cleaning
- Exploratory Data Analysis
- Feature Selection
- Train-Test Split
- Linear Regression Model
- Prediction
- Model Evaluation

## Model Performance
- **Mean Absolute Error (MAE):** 1.4608
- **Mean Squared Error (MSE):** 3.1741
- **R² Score:** 0.8994

## Files Included
- `sales_prediction.py`

## Conclusion
The Linear Regression model successfully predicted sales with high accuracy. The project demonstrates a complete machine learning workflow from data preprocessing to model evaluation.
