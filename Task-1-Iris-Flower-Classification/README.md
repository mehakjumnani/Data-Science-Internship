# Iris Flower Classification

## Objective
Build a machine learning model to classify Iris flowers into Setosa, Versicolor, and Virginica based on their flower measurements.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
- Iris Dataset (iris.csv)

## Algorithm
- Decision Tree Classifier

## Project Workflow
1. Loaded the dataset.
2. Explored the data.
3. Checked missing values.
4. Performed data visualization.
5. Split the dataset into training and testing sets.
6. Trained the Decision Tree model.
7. Predicted flower species.
8. Evaluated the model using Accuracy Score and Confusion Matrix.

## Result
- Accuracy: **100%**
